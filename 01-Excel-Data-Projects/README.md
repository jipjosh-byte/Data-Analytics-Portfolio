# Student Performance & Academic Achievement Analysis (Excel)

## 📌 Project Overview
- **Objective:** To clean, aggregate, and evaluate classroom grade tracking data to quickly identify academic performance tiers, student strengths, and subject-matter outliers.
- **Organization Type:** Educational Institution / Academy / School Board Trust[cite: 2]
- **Business Value:** Enables educators and administrators to instantly spot at-risk students needing intervention, reward top performers, and assess teacher/curriculum efficacy across subjects[cite: 2].

## 🛠️ Tools & Advanced Functions Used
- **Excel Tables (`Ctrl + T`):** Formatted for dynamic data management[cite: 3].
- **Statistical Functions:** Used `=AVERAGE()` for overall grading, `=MAX()` for peak potential detection, and mathematical ranges to analyze performance spreads.
- **Logical Operations:** Built custom `=IF()` / evaluation rules to automate "Pass/Fail" states (Threshold: 75)[cite: 3].
- **Data Categorization:** Deployed standard nested evaluations to assign qualitative descriptors ("Excellent", "Good", "Average")[cite: 3].
- **Conditional Formatting:** Applied data-driven color scaling to highlight the highest and lowest averages instantly[cite: 3].

## 📊 Dataset Used
- **Source:** LeapTalent Data Bootcamp (Kaggle-derived academic matrix)[cite: 2]
- **Description:** Contains individual testing outcomes for student cohorts across core modules: English, Mathematics, and Science[cite: 3].

## 🚀 Analytical Steps Taken
1. **Database Conversion:** Converted raw data structures into a formalised Excel Table named `Student_grades_table` for dynamic filtering[cite: 3].
2. **Subject Performance Auditing:** Sorted and isolated maximum scores per discipline to identify top subject specialists[cite: 3].
3. **Descriptive Statistics Calculation:** Formulated baseline classroom performance tracking metrics (Averages, Max Potential, Performance Ranges)[cite: 2, 3].
4. **Automated Visual Logic:** Engineered automated scoring columns to classify achievement tiers without manual entry overhead[cite: 3].
5. **Exception Highlighting:** Deployed automated heat mapping via conditional rules to guide stakeholders' eyes directly to low-performing exceptions [cite: 3].

## 💡 Key Findings & Insights (Soft Skills)
- **Finding 1:** Quantitative analysis identified Carol as the top student classroom-wide with an overall metric of 81.67, driven by strong consistency across Science and Mathematics (85 each)[cite: 3].
  - *Business Meaning:* Spotlighting top performers allows administrators to map out curriculum frameworks based on successful student profiles[cite: 2].

  - *Business Meaning:* This immediate visual flag tells administration exactly where localized tutoring resources must be deployed before terminal testing intervals[cite: 2].
